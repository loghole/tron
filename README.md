# Tron

### Installation

##### install tron: 

```
curl -sSfL https://raw.githubusercontent.com/loghole/tron/master/install.sh | sh
tron version
```

##### install protoc: 

* linux
 ```
$ apt install -y protobuf-compiler
$ protoc --version
```

* mac:
```
$ brew install protobuf
$ protoc --version
```
